dist文件夹为源文件被解析后的文件并且进行了压缩
node_modules文件夹为下载的开发依赖和运行依赖的插件
public文件夹为该项目的入口文件，也是公共文件
src文件夹为该项目的源文件
src中的文件夹：1.assets:是一些图片和一个css样式的重置
              2.components中文件夹:1.common文件夹为一些公共我自行封装的一些组件
                                   2.comtent文件夹为在该项目下运用的我自行封装的组件
              3.network文件夹中的代码主要负责进行网络请求的代码
              4.router文件夹中的代码为路由对网页跳转进行封装的
              5.store文件夹中的代码为vuex的部署
              6.views文件夹为页面实现的代码
